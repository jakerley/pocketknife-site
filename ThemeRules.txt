Typography
* Font: Inter (preferred), fallback to system UI fonts.
* Font weights: Use 400 (normal), 500 (medium), 700 (bold) as needed.
* Font sizes:
* Titles: 18–22px
* Body: 15–16px
* Captions/hints: 13–14px
UI Style Guidelines
* Corners: Use 12px border-radius for cards, modals, and main app container.
* Shadows: Subtle, e.g., box-shadow: 0 8px 32px rgba(0,0,0,0.4);
* Buttons:
* Primary: Solid primary color, white text, rounded corners.
* Secondary: Transparent or surface background, border, textSecondary color.
* Hover: Use primaryHover or lighten/darken as appropriate.
* Inputs:
* Background: surface
* Border: border
* Text: text
* Focus: Use primary for border or shadow.
* Spacing:
* Use 16–24px padding for main containers.
* 8–16px spacing between elements.
* Icons:
* Use simple, monochrome icons (white or textSecondary).
* Prefer SVG or icon fonts.
* Dark Mode:
* The above palette is dark mode by default. If you add light mode, invert appropriately.
